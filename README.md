# KenKeeper

## Overview
KenKeeper is a web application that helps users track their friendships by monitoring the last time they connected with each friend. It provides a simple and visual way to understand communication history and maintain stronger relationships.

---

## Technologies Used
- Next.js  
- React  
- Tailwind CSS  
- JavaScript  
- Chart.js (or any chart library used for visualization)

---

## Key Features

### 1. Friend Details Page
Clicking the **Details** button navigates to a dedicated page where you can view complete information about a specific friend.

### 2. Interaction History Tracking
Every interaction (call, message, video, etc.) is stored in a history/notification system, allowing users to track communication over time.

### 3. Visual Analytics with Charts
The app visualizes friendship interaction data using charts, making it easy to analyze communication frequency and patterns.
